# Artifact Runtime

Pengelola dokumen semantik, klasifikasi, lineage, dan proyeksi pengetahuan untuk AetherOS.

## Struktur Folder
- `src/aether_artifact/core/`: Model domain artifact, lineage, dan projection.
- `src/aether_artifact/classification/`: Validasi tipe dokumen semantik.
- `src/aether_artifact/registry/`: Pendaftaran global artifact.
