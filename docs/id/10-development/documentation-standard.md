# Documentation Standard

1. **Docstrings**: Setiap class, fungsi, dan metode publik WAJIB memiliki docstring.
2. **README Package**: Setiap direktori/package besar wajib memiliki `README.md` yang menjelaskan tanggung jawab, aturan, dan contoh penggunaan.
3. **Architecture Decision Records (ADR)**: Setiap perubahan pada Contract (Public API) atau desain Kernel harus didahului dengan ADR.
4. **Diagram**: Gunakan *Mermaid* untuk Sequence Diagram dan Dependency Graph dalam dokumentasi arsitektur.
