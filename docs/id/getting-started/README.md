# Development Onboarding & Standards

Direktori ini berisi seluruh panduan memulai (*onboarding*), aturan baku pengkodean, standar pengetesan, dan checklist kelayakan rilis untuk repositori AetherOS.

## Daftar Panduan

1. **[Developer Quickstart Guide](quickstart.md)**: Langkah instalasi lokal, dependensi `uv`, dan cara menjalankan unit testing.
2. **[Runtime Package Checklist](checklist.md)**: Kriteria kepatuhan arsitektur (ADR-0027) untuk pembekuan API.
3. **[Kernel Development Rules](kernel-development-rules.md)**: Aturan modifikasi lapisan kernel dasar.
4. **[Coding Standard](coding-standard.md)**: Panduan penulisan kode Python bersih.
5. **[Dependency Rules](dependency-rules.md)**: Aturan impor antar-paket.
6. **[Testing Standard](testing-standard.md)**: Standar uji coba program.
7. **[Documentation Standard](documentation-standard.md)**: Format penulisan spesifikasi.
