# ADR-0002: Use Constructor Injection

## Status
Accepted

## Context
Use Pure Python Constructor Injection to decouple dependencies without third-party heavy DI libraries.

(Keputusan implementasi spesifik terkait 0010-use-constructor-injection.md)
